# Donation Bar
A simple, manually controlled, donation progress bar. Made for EdinburghsNicestBoy.

**Note: This bar does not remember it's state if the browser source or page are refreshed**

## Usage

Use the following three commands to control the bar. Whole and decimal amounts are possible.

`!donogoal 1000.42` - Set the goal amount

`!donoadd 100.42` - Add a donation

`!donominus 100.42` - Remove a dontation
